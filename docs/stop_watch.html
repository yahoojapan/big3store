Description

A server for recording lap events and report elapsed time. This module is implemented as an erlang gen_server process.

handle_call (synchronous) message API

(LINK: handle_call (synchronous) message API)

{start, Label}

Starts the stopwatch with Label::string(). It returns {started, Label::string(), LocalTime::calendar:datetime1970()}}. This request is implemented by hc_start/2. (LINK: {start, Label})

{record, Label}

Records the event with Label::string(). It returns {recorded, Label::string(), LocalTime::calendar:datetime1970()}. This request is implemented by hc_record/2. (LINK: {record, Label})

report

Reports the number of request and elapsed time after receiving {start, Label} request. It returns {{number_of_rec, Num::integer()}, {elapsed_microsec, Elapsed::integer()}}. This request is implemented by hc_report/1. (LINK: report)

handle_cast (asynchronous) message API

(LINK: handle_cast (asynchronous) message API)

{record, Label} (cast)

Records the event with Label::string(). This request is implemented by hc_record/2. (LINK: {record, Label} (cast))
